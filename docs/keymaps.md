# Горячие клавиши

Leader = `Пробел`

---

## Навигация (Vim)

| Клавиша | Описание |
|---------|----------|
| `h j k l` | Влево, вниз, вверх, вправо |
| `w / b` | Слово вперёд / назад |
| `e / ge` | Конец слова вперёд / назад |
| `0 / ^ / $` | Начало / текст / конец строки |
| `gg / G` | Начало / конец файла |
| `{ / }` | Параграф вверх / вниз |
| `%` | К парной скобке |
| `H / M / L` | Верх / середина / низ экрана |
| `Ctrl+u / Ctrl+d` | Половина страницы вверх / вниз |
| `zz` | Центрировать курсор |
| `f{char}` | Найти символ в строке |
| `t{char}` | Найти символ до |
| `; / ,` | Повторить f/t вперёд / назад |

---

## Редактирование (Vim)

| Клавиша | Описание |
|---------|----------|
| `i / a` | Вставка перед / после курсора |
| `I / A` | Вставка в начало / конец строки |
| `o / O` | Новая строка ниже / выше |
| `cc` | Заменить всю строку |
| `cw / c$` | Изменить слово / до конца строки |
| `r / R` | Заменить символ / режим замены |
| `x / X` | Удалить символ под / перед курсором |
| `dd / yy` | Удалить / скопировать строку |
| `dw / yw` | Удалить / скопировать слово |
| `p / P` | Вставить после / перед курсором |
| `u / Ctrl+r` | Отменить / повторить |
| `.` | Повторить последнее действие |
| `Ctrl+a / Ctrl+x` | Инкремент / декремент числа |
| `~` | Изменить регистр символа |

## Визуальный режим

| Клавиша | Описание |
|---------|----------|
| `v` | Символьное выделение |
| `V` | Построчное выделение |
| `Ctrl+v` | Блочное выделение |
| `> / <` | Сдвинуть вправо / влево |
| `y / d` | Копировать / удалить выделение |
| `J / K` | Переместить выделенное вниз / вверх |
| `o / O` | Переместить курсор в другое ребро |
| `gv` | Повторить последнее выделение |

## Поиск и замена

| Клавиша | Описание |
|---------|----------|
| `/pattern` | Поиск вперёд |
| `?pattern` | Поиск назад |
| `n / N` | Следующее / предыдущее совпадение |
| `*` | Искать слово под курсором |
| `:%s/old/new/g` | Заменить всё |
| `:%s/old/new/gc` | Заменить с подтверждением |
| `:noh` | Убрать подсветку поиска |
