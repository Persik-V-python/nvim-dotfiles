# Описание плагинов

## Внешний вид

### catppuccin
**Тема оформления.** Красивая тёмная тема с палитрой Mocha. Интегрируется со всеми основными плагинами: cmp, treesitter, telescope, gitsigns и др.

### nvim-web-devicons
**Иконки файлов.** Набор иконок для различных типов файлов. Используется nvim-tree, bufferline, lualine и другими плагинами.

### nvim-tree.lua
**Файловый менеджер.** Древовидный проводник файлов с иконками, git-индикаторами и быстрыми действиями. Открывается по `<leader>e`.

### bufferline.nvim
**Вкладки буферов.** Отображает открытые буферы в виде вкладок с номерами, диагностикой и git-статусом.

### lualine.nvim
**Статусная строка.** Нижняя панель с режимом, веткой git, диагностикой, именем файла, кодировкой и позицией курсора.

### alpha-nvim
**Стартовый экран.** Показывает ASCII-логотип Neovim и кнопки быстрого доступа при открытии без файла.

### nvim-notify
**Уведомления.** Красивые всплывающие уведомления вместо стандартных сообщений vim.

### which-key.nvim
**Подсказки по клавишам.** Показывает доступные команды при нажатии префикса (например, `<leader>`).

### indent-blankline.nvim
**Индентация.** Визуальные линии для отображения уровней отступов.

### nvim-colorizer.lua
**Цвета.** Подсвечивает hex-коды цветов (#ff0000 → красный квадрат) и другие форматы прямо в коде.

## Навигация

### telescope.nvim
**Fuzzy Finder.** Мощный поиск по файлам, тексту, буферам, символам и командам. Интеграция с fzf-native для скорости.

### telescope-fzf-native.nvim
**Расширение Telescope.** Нативное расширение fzf на C для максимальной производительности поиска.

### flash.nvim
**Быстрый jump.** Позволяет мгновенно перемещаться к любому символу на экране. `s` для jump, `S` для treesitter.

### neoscroll.nvim
**Плавная прокрутка.** Плавная анимация при прокрутке страниц (Ctrl+u, Ctrl+d, gg, G).

### smart-splits.nvim
**Умные сплиты.** Улучшенная навигация между окнами Neovim и tmux.

## Редактирование

### nvim-treesitter
**Парсер синтаксиса.** Современная подсветка синтаксиса, автоотступы, авто-теги и инкрементальное выделение.

### nvim-autopairs
**Автоскобки.** Автоматически закрывает скобки, кавычки и другие пары символов.

### nvim-surround
**Окружение.** Добавление, изменение и удаление скобок/кавычек вокруг текста. `ysiw"` — обернуть слово в кавычки.

### Comment.nvim
**Комментарии.** Быстрое комментирование строк. `gcc` — строка, `gc` — визуальный режим.

### todo-comments.nvim
**TODO-комментарии.** Подсвечивает TODO, FIXME, HACK и другие маркеры. Интеграция с Telescope и Trouble.

### trouble.nvim
**Список ошибок.** Удобный интерфейс для просмотра диагностик, quickfix и локаций.

## LSP и IntelliSense

### nvim-lspconfig
**Конфигурация LSP.** Настройка Language Server Protocol для автодополнения, навигации, рефакторинга и диагностики.

### mason.nvim
**Установщик LSP.** Графический интерфейс для установки LSP-серверов, линтеров и форматтеров.

### mason-lspconfig.nvim
**Мост Mason ↔ LSPConfig.** Автоматическая настройка LSP-серверов, установленных через Mason.

### nvim-cmp
**Автодополнение.** Контекстное автодополнение с источниками: LSP, буфер, путь, LuaSnip.

### LuaSnip
**Сниппеты.** Движок сниппетов с поддержкой VS Code формата через friendly-snippets.

### friendly-snippets
**Готовые сниппеты.** Коллекция сниппетов для множества языков программирования.

### none-ls.nvim
**Форматирование и линтинг.** Интеграция внешних форматтеров (prettier, black, stylua) и линтеров (eslint_d).

## Git

### gitsigns.nvim
**Git-индикаторы.** Показывает изменения в файле прямо в номерах строк: добавления, изменения, удаления. Stage, reset, preview hunks.

## AI и Терминал

### codock.nvim
**Интеграция AI CLI.** Открывает вертикальный сплит с AI CLI (Crush, Claude, OpenCode и др.). Поддерживает копирование позиции файла для контекста.

## Kimi Usage (кастомный плагин)

**Мониторинг лимитов Kimi.** Каждые 2 минуты опрашивает API Kimi (`api.kimi.com/coding/v1/usages`) и показывает плавающее окно с ASCII прогресс-барами:

- Недельный лимит использования
- 5-минутное окно (300 сек)
- Оставшаяся квота

Требует переменную окружения `KIMI_API_TOKEN` или `g:kimi_api_token`.

## Сессии

### persistence.nvim
**Управление сессиями.** Автоматически сохраняет и восстанавливает сессии редактирования при открытии/закрытии Neovim.
