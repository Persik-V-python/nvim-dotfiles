📖 Мини-гайд: Как пережить первую неделю

1. Режимы — главное отличие

  • Normal — для навигации и команд
  • Insert — для ввода текста (i, a, o)
  • Visual — для выделения (v, V, Ctrl+v)
  • Command — для команд (:w, :q)
💡 Всегда возвращайся в Normal режим (Esc). Это домашняя позиция.

2. Мышь не нужна

  • h j k l — базовое перемещение
  • w b e — по словам
  • f{char} / t{char} — найти символ в строке
  • gg / G — начало / конец файла
  • { / } — по параграфам
💡 Скорость придёт через 3-5 дней активного использования.

3. Объектно-глагольная модель

  • d — delete (удалить)
  • y — yank (копировать)
  • c — change (изменить)
  • v — visual (выделить)
💡 d2w = delete 2 words. ci" = change inside quotes. yy = yank line.

4. Text objects — суперсила

  • iw / aw — inner / around word
  • i" / a" — inner / around quotes
  • i( / a( — inner / around parentheses
  • it / at — inner / around tag (HTML)
💡 ci" изменит текст внутри кавычек. da( удалит скобки и всё внутри.

5. Макросы для повторяющихся задач

  • qa — начать запись макроса в регистр a
  • q — остановить запись
  • @a — воспроизвести макрос a
  • @@ — повторить последний макрос
  • 10@a — выполнить 10 раз
💡 Это как multi-cursor, но мощнее. Идеально для форматирования JSON.

6. Что важно помнить

  • . — повторить последнее действие
  • ; / , — повторить f/t вперёд / назад
  • Ctrl+a / Ctrl+x — инкремент / декремент числа
  • >> / << — отступ для строки
  • ~ — изменить регистр символа
  • Ctrl+r = — посчитать выражение (в insert)
💡 Не пытайся выучить всё сразу. Начни с h j k l, i, Esc, :w, :q.
